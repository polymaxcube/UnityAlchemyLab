# 🧪 UnityAlchemyLab

**Turning scripts into systems.**

A development lab for crafting reusable Unity systems through experimentation — where gameplay mechanics, simulation logic, and tools are refined into clean, modular code.

---

## ✨ Features

* Modular and reusable Unity scripts
* Clean and scalable architecture
* Gameplay systems (movement, camera, interaction)
* Simulation-focused components (motion, constraints)
* Developer utilities and tools

---

## 📦 Installation

### Option 1 — Copy

Clone or download this repository and copy the scripts into your Unity project.

### Option 2 — Git URL (UPM)

```bash
https://github.com/your-username/UnityAlchemyLab.git
```

---

## 📁 Project Structure

```bash
UnityAlchemyLab/
├── Runtime/
│   ├── Core/               # Base systems (Singleton, EventBus, Services)
│   ├── Gameplay/           # Movement, camera, interaction
│   ├── Simulation/         # Motion, constraints, physics systems
│   ├── Systems/            # Health, inventory, reusable logic
│   └── Utilities/          # Math, helpers, extensions
│
├── Editor/                 # Custom Unity Editor tools
├── Samples~/               # Example scenes and usage
├── Documentation~/         # Guides and architecture notes
├── Tests/                  # Optional test cases
│
├── package.json
├── README.md
└── LICENSE
```

> The project follows a modular, package-oriented structure inspired by Unity Package Manager conventions.

## 🧠 Philosophy

Inspired by the concept of **alchemy** — combining simple components into powerful systems.
Each script is designed to be flexible, reusable, and easy to integrate.
